**WisteRia** is a premium, AI-powered travel planning application built for power users who want total control over their itineraries without the manual data entry headache.

![WisteRia Hero](landing/assets/hero.png)

## ✨ Key Features

- **AI Magic Box**: Paste messy text, group chat snippets, or links. Our AI parses and maps them directly to your timeline.
- **Visual Versioning**: Create "branches" of your trip (e.g., "Budget Option", "Luxury Swaps"). Merging choices is as easy as git.
- **Interactive Timeline**: A rich, drag-and-drop canvas for precision scheduling with collision detection and "free time" analysis.
- **Universal Context**: Attach PDFs, tickets, and screenshots directly to events. Your docs are always where you need them.
- **Privacy First**: Fully local SQLite storage. Your data never leaves your machine.

## 🚀 Getting Started

### Prerequisites

- macOS (Intel or Apple Silicon)
- Apple Silicon recommended for local AI features

### Installation

1. Download the latest `.dmg` from the [Releases](https://github.com/SukeshSiva/WisteRia/releases) page.
2. Drag **WisteRia.app** to your Applications folder.
3. Open and start planning!

## 🛠️ Development

WisteRia is built with **Tauri**, **React**, and **Rust**.

```bash
# Clone the repository
git clone https://github.com/SukeshSiva/WisteRia.git

# Install dependencies
npm install

# Run in development mode
npm run tauri dev
```
