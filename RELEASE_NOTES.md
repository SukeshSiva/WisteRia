# WisteRia Beta Release v0.1.0-beta.1

Welcome to the first beta release of WisteRia! This release focuses on the core trip planning experience with a beautiful macOS interface.

## New Features
- **Dynamic Timeline View**: Organize your trips with a visual timeline that helps you see your itinerary at a glance.
- **AI-Powered Trip Planning**: Seamlessly generate and manage trip details with integrated AI support.
- **Mac-Native Experience**: Custom UI elements including native dock icon switching and macOS-specific styling.
- **Enhanced PDF Appendix**: Restructured PDF exports to correctly interleave and display attachments.
- **Undo/Redo System**: Full support for undoing and redoing changes to your trip itinerary.

## Bug Fixes & Improvements
- **Fixed App Crashes**: Resolved several stability issues during app startup and timeline rendering.
- **Performance Optimizations**: Improved response times for timeline updates and cost breakdowns.
- **UI Polish**: Standardized color schemes and improved responsiveness for smaller windows.

## Known Issues
- This is a beta release; minor UI glitches and performance hiccups may still occur.
- Please report any bugs via GitHub issues.
